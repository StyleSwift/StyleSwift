🔍 重复代码检测测试

📋 功能说明

StyleSwift 现在具备智能重复代码检测功能，避免在数据库中存储相同的样式和挂件代码，提高存储效率和系统性能。

🚀 优化机制

  • SHA256哈希算法：对代码内容进行标准化后计算哈希值
  • 智能去重：保存前检查是否存在相同哈希值的代码
  • 引用复用：发现重复时返回现有记录的ID，避免重复存储
  • 空间优化：显著减少数据库存储空间占用

🧪 测试步骤

步骤 1: 测试样式重复检测

1. 在 StyleSwift 扩展中创建一个自定义样式

2. 记录返回的样式ID

3. 再次提交完全相同的样式代码

4. 验证是否返回相同的样式ID

5. 检查浏览器控制台是否显示"检测到重复样式"

步骤 2: 测试挂件重复检测

1. 创建一个自定义代码挂件（HTML + CSS + JS）

2. 记录返回的挂件ID

3. 再次提交完全相同的挂件代码

4. 验证是否返回相同的挂件ID

5. 检查浏览器控制台是否显示"检测到重复挂件"

步骤 3: 测试AI生成去重

1. 使用相同的描述生成AI样式或挂件

2. 如果AI生成了完全相同的代码

3. 系统应该自动检测并复用现有代码

4. 观察是否有重复检测的提示信息

🛠️ 技术实现细节

哈希计算算法

def calculate_code_hash(content):
    # 标准化代码内容
    normalized = re.sub(r'\s+', ' ', content.strip())
    # 计算SHA256哈希
    return hashlib.sha256(normalized.encode()).hexdigest()

重复检测逻辑

# 检查是否已存在相同的代码
existing = Style.query.filter_by(
    style_code_hash=style_hash
).first()

if existing:
    return existing.style_id  # 复用现有

⚠️ 注意事项

  • 哈希计算会标准化代码格式，轻微的空格差异不会影响检测
  • 只有代码内容完全相同才会被识别为重复
  • 重复检测不会影响样式或挂件的功能
  • 系统会在控制台提供详细的检测日志
