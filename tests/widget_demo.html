📦 数据库中的挂件信息

Widget 数据表结构：

widget_id: 'catgirl' | 'transformer' | 'pokemon'
name: 挂件显示名称
description: 挂件功能描述
widget_type: 枚举类型 (catgirl/transformer/pokemon/custom/custom-code)
html_code: 完整的HTML结构代码
css_code: 样式定义和动画效果
js_code: 交互逻辑和功能实现
default_config: JSON格式的默认配置 (位置、大小等)
created_at: 创建时间
rating: 评分系统字段

API 端点:
GET  /api/widgets - 获取所有挂件列表
POST /api/apply_widget - 应用预设挂件
POST /api/generate_ai_widget - 生成自定义挂件
POST /api/save_custom_widget_code - 保存自定义代码挂件
